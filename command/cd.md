# cd

cd 命令用于切换当前工作目录至目标目录

目标目录可以是*绝对路径*或*相对路径*。 若目录名称省略，则切换至当前用户的home目录。

另外，`~` 也表示为当前用户home目录，`.` 表示当前所在目录，`..` 表示当前目录上一级目录，`-` 表示上一次所在目录。

## 语法

```bash
cd [目标目录]
```

## 实例

切换至/usr/bin/

```bash
cd /usr/bin
```

切换到当前用户home目录

```bash
cd ~
```

切换到当前目录的上两层目录

```bash
cd ../..
```

切换回上一次所在目录

```bash
cd -
```
